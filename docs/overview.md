# 项目与工具概览

这个仓库是一个 FastAPI + PostgreSQL + Redis 的后端模板，使用 Docker Compose 管理依赖（db/redis/web）。

## 关键工具

- **Python 3.11**：编程语言
- **FastAPI**：web 框架
- **Pydantic**：配置与数据校验
- **Uvicorn**：ASGI 服务器
- **pytest**：测试框架
- **docker / docker compose**：容器化
- **git / GitHub**：版本控制与远程托管
